Blockchain / NFT Minting

Controlled collectible minting, never default exposure.

Minting is handled as a controlled production step and follows package policy, approval controls, and runtime checks.

Minting Scope

  • Minting is a separate production step unless explicitly included in package scope.
  • Collectible minting covers preparation, metadata setup, mint execution, and applicable network fees.
  • Private vault materials are not minted by default.
  • Only approved delivery-safe collectible assets are eligible for minting.

Launch Controls

  • Minting is never implied as automatic at checkout, upload, or intake.
  • Automatic mint execution is not live by default and requires verified runtime enablement.
  • Scheduled reveal automation is planned and currently private beta / in development.
  • Automatic timed release is not live without a dedicated production executor.

Public-Proof Boundaries

  • Public layers are limited to approved collectible artifacts and proof metadata.
  • Private household and organization workspaces remain protected and role-scoped.
  • Package eligibility and approvals determine whether a collectible proceeds to minting.
  • Organization-lane anchors remain opt-in and policy-controlled.
Maintenance

Continuity support for delivered projects.

Maintenance is a separate purchase and applies to sustaining the delivered project inside the package scope you selected.

Maintenance Covers

  • Storage continuity and protected access continuity
  • Retrieval support for delivered project materials
  • Role-based workspace continuity for approved members
  • Package-compatible platform updates and delivery continuity
  • Normal support for the delivered project

Maintenance Does Not Cover

  • A new build or a new project outside delivered scope
  • New branch creation that was not purchased
  • Genealogy research services
  • Major redesign or re-architecture work
  • Add-ons or extras not purchased

Plan Activation

  • Maintenance starts on delivery or activation based on package policy.
  • Monthly, annual, and lifetime options are available by package.
  • Maintenance does not replace upgrade pricing or add-on purchases.
  • Full package upgrades remain available by paying the package difference.
