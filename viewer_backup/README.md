# Tomb of Light – Viewer

This is the interactive NFT family tree viewer for the Tomb of Light project.

## Project Roles
- **Larry Robinson** – Founder, backend owner
- **Leon Floyd** – Frontend collaborator (HTML/CSS/JS)

## How to Run It
1. Clone the repo
2. Open `index.html` in any browser
3. Interact with the image using zoom gestures

Frontend-only. No server setup required.
