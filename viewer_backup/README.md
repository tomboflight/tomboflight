# Tomb of Light

Tomb of Light is a digital legacy platform designed to preserve family lineage, memory, and identity across generations.

The platform allows users to explore a living family tree through interactive portraits. By zooming into or out of an image, users can travel through generations — revealing parents, descendants, and sibling branches.

This repository contains the **Genesis Prototype Viewer**, which demonstrates how the Tomb of Light lineage navigation system works.

---

# Genesis Prototype Viewer

The Genesis Prototype is the first working demonstration of the Tomb of Light interactive family tree.

Users begin at a central portrait and can:

• Zoom out to reveal parents  
• Zoom in to reveal descendants  
• Navigate through generations  
• Explore sibling branches

All navigation occurs through zoom gestures rather than traditional menus.

# Running the Viewer

Clone the repository:

git clone https://github.com/tomboflight/tomboflight.git

Then open the viewer in your browser:

viewer/index.html

No server is required.  
The viewer runs directly in the browser.
